# Konquest Boats

Marketing website for **Konquest Boats** — a builder of inflatable Zodiacs and pontoon boats. *Built to conquer the water.*

## What's here

- `index.html` — the webpage (one page, with sections for the boats, the story, and a quote request form)
- `styles.css` — the look and feel (colors, fonts, layout)

## How to view it

Open `index.html` in any web browser, or ask Claude to "run the Konquest site" and it'll show a live preview.
